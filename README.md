# Resume Parser - Firefox Extension

A Firefox browser extension that parses resume text and autofills job application forms.

##Feature
- Paste your resume text and extract sections automatically
- Adding card to each of the section in your resume
- Categorizes resume sections (skills, education, experience, etc).
- Fill website with your resume to save time from manually type.

##How to install
- Clone the repo at the upright top of the github repo
- After download, went to the FireFox url "about:debugging"
- Go to This FireFox
- Go to Load Temporary-Add on...
- Open the folder and click the manifest.json (to let browser know this is the setup) 

##How to use
- Copy and Paste your Resume to the text area
- Click Extract Resume to verify your resume's information
- Click Fill Form when everything is correct and fill your dream job website.
